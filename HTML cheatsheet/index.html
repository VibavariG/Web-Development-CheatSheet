HTML Reference Cheatsheet

Basic HTML Tags

Tag Attributes Name Description
<p> Paragraph Defines a Paragraph
<div> class, id Division Divides off a section of code
<a> target, href Anchor Used to link to other locations, target="_blank" for new tab
<ul> Unordered list Lists elements in bullet points
<ol> Ordered list Lists elements in numbers or some ordered way
<li> List Used to define every item in an ordered or undordered list
<span> Span To apply some CSS to a very small part of text.
<em> Emphasize Makes text italicized
<strong> Strong Makes text bold
<br> Break Works like endl

Multimedia Tags

Tag Attributes Name Description
<img> src, alt Image Adds an image, self closing.
<video> src, controls, loop, autoplay Video Adds a video, content will be diplayed if video not visible
<audio> controls, autoplay Audio Inserts audio file
<source> src, type="audio/mp3" Source Adds source for audi, used inside <audio>
<embed> src Embed Can embed any media content

Table Tags

Tag Attributes Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Row Defines an entire row of the table
<th> rowspan, colspan Head Usually the column heads in a table. Bold
<td> rowspan, colspan Table Data Cell contents
<tfoot> Table Footer Contains the footer of the table

Form Tags

Tag Attributes Name Description
<form> action, method Form Wrapper element for all forms, action- where to send, method- way of sending
<label> for Label To properly identify an input, for = id of input
<input> type, name, id, value Input Tpye takes values-
password
range (min, max, step)
submit
text (minlength, maxlength, pattern)
number (min, max)
radio
checkbox
<textarea> rows, cols Input To accept a multiple line input

Semantic Tags

Tag Attributes Name Description
<header> Header Defines a header
<footer> Footer Defines a Footer
<nav> Navigation Defines a Menu
<main> Main Defines Content of page
<article> Article Content that makes sense on its own
<section> Section Content with the same theme
<aside> Aside Additional information
<figure> Figure Encapsulates media - image
<figcaption> Figure Caption Inside <figure>, defines the caption of the image
